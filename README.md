# Quickselect Visualizer

An interactive visualization tool demonstrating the Quickselect algorithm, along with a paradigm analysis using Divide and Conquer with a greedy approach.

## Live Demo
🔗 https://leannejrodrigo.github.io/quickselect-visualizer/

## Features
- Visualizes Quickselect step-by-step
- Shows pivot selection and partitioning
- Helps understand recursion and selection process
